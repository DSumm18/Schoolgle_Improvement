# Navigator MVP Spec

## MUST DO
- Tenant separation (RLS)
- Role-based access
- Pack approvals lock versions
- A4 PDF exports
- Timeline entries on pack export/SOP complete
- Audit trail

## MUST NOT DO
- Chrome extension
- Autopilot/form submission
- Deep integrations
- Pupil-level SEND timelines
- AI auto-fill without consent

## VERSIONING RULES
Create version on: manual save, section complete, submit, approval, export
Do NOT version on every keystroke
