# One-Click Reports

## Generate Inspection-Ready Documents in Seconds

### The Problem

Schools spend **weeks** every year writing the same statutory documents:
- SEF takes 3-4 weeks to write
- Pupil Premium Strategy needs annual updates
- Sports Premium Report required by DfE
- SDP reviewed and rewritten annually

**Result:** 
- Stale documents (SEF "last updated 8 months ago")
- Copy-paste from previous years
- Inconsistent messaging
- Time stolen from actual improvement work

### The Solution

One click. 45 seconds. Done.

1. School overview
2. Funding overview
3. Part A: Pupil Premium Strategy Plan
   - Statement of intent
   - Challenges
   - Intended outcomes
   - Activity (Teaching, Targeted, Wider)
4. Part B: Review of Previous Year

### Report Quality Indicators

Each generated report shows:
- **Completion percentage** - How much data was available
- **Section status** - Complete / Partial / Missing data
- **Missing data alerts** - What to add for better output
- **Word count** - For reference

### Export Options

- **HTML** - For printing, formatted beautifully
- **Markdown** - For further editing
- **Copy to clipboard** - Paste anywhere
